Documentation

Needlessly Small Rod (NSR) was created for the Riot API Challenge 2.0, which took place from August 10 - August 31, 2015. NSR was created by Joseph 'Tanglewreak' Bonilla and Elizabeth 'Lizabeth' Duong.

The purpose of NSR is to demonstrate the changes in AP item usage by showing players the best builds for each champion and lane, both before and after patch 5.13 of the AP item changes.

Upon visiting the main page, users are tested with a question to test their knowledge of the AP item changes that occurred in Patch 5.13. If the question is answered correctly, the user is allowed access to the page where they can choose a champion and lane to find builds and stats for. Otherwise, the user is sent to a page documenting the AP item changes.

Builds are obtained based on the given set of matches, common builds, and highest winrate for each lane. AD Carry and Support are considered Bot lane. Matches are from normals and ranked Summoner's Rift from both patch 5.11 and patch 5.14. Some of the provided suggested builds may seem unconventional; However, this is due to some of the restrictions and accuracy of data from the Riot API.

Data from the Riot API was obtained between 8/10-8/31, 2015. How we obtained this data was through the use of development keys provided by Riot. We acquired four types of information: normal and ranked match data from patch 5.11 and patch 5.14.
